DataTable supports row detail view through the `RowDetailPlugin` from `@izumisy/seizen-datatable-plugins`.

## Basic Usage

```tsx
import { useDataTable, DataTable, useDataTableEvent } from "@izumisy/seizen-datatable-react";
import { RowDetailPlugin } from "@izumisy/seizen-datatable-plugins/row-detail";

function MyTable() {
  const table = useDataTable({
    data,
    columns,
    plugins: [RowDetailPlugin.configure({ width: 350 })],
  });

  // Open sidepanel when a row is clicked
  useDataTableEvent(table, "row-click", (row) => {
    table.plugin.open("row-detail", { row });
  });

  return <DataTable table={table} />;
}
```

## How It Works

1. User clicks on a table row
2. The `row-click` event is emitted
3. Your event handler opens the row-detail plugin with the clicked row
4. The sidepanel displays all fields of the row

## Features

- **Auto-update** - Clicking a different row while the panel is open updates the content
- **JSON formatting** - Object and array values are formatted as JSON
- **Scroll support** - Long content is scrollable within the panel

## Configuration Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `width` | `number` | `320` | Width of the sidepanel |

## Opening Programmatically

You can open the row detail panel programmatically for any row:

```tsx
// Open for a specific row
table.plugin.open("row-detail", { row: myRowData });

// Close the panel
table.plugin.close("row-detail");
```

<Aside type="note">
  The `useDataTableEvent` hook is used to subscribe to table events. The `row-click` event passes the entire row data object to your handler.
</Aside>
