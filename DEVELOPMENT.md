# Development

## Prerequisites

- Node.js 20+
- pnpm 9+

## Setup

```bash
# Install dependencies
pnpm install
```

## Commands

```bash
# Build all packages
pnpm build

# Run tests
pnpm test

# Lint
pnpm lint
```

## Project Structure

```
seizen-ui/
├── packages/
│   ├── datatable-react/          # DataTable component
│   └── command-palette-react/    # Command palette component
└── examples/                     # Example applications
```
